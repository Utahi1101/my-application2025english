# my-application2025
英単語をランダム出力する英単語帳

# 背景


# 機能の概要				
・単語の品詞、ジャンル、レベルを出力

・出力単語の関連または類義語の出力

・単語によっては画像も同時に出力

・単語のブックマーク機能、履歴機能

・固有名詞の出力の有無の設定

・各単語の出力回数の表示

# 使用した技術（AIなど）
ChatGPT、Google Gemini
利用API:
- Random Word API (https://random-word-api.herokuapp.com/)
- Dictionary API (https://dictionaryapi.dev/) [CC BY-SA 3.0]
- LibreTranslate (https://libretranslate.com/) [AGPL v3]
